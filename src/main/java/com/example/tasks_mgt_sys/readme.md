# A simple multi-tenant Task & Project Management System that supports multiple organizations, users, and role-based access (RBAC). 

TESTING
CLI interface (for admin/testing)
REST API (for real-world usage) 

CORE FEATURES
- JWT(Json Web Token)-based authentication
- Shell for CLI(Command Line Interface)
- Reddis for OTP caching
- Email notification simulation
- MVC (Model View Controller) and Clean Architecture
- Audit Logs
- Pagination and Filtering
- Soft delete

Published documentation: http://localhost:8093/swagger-ui/index.html#/
